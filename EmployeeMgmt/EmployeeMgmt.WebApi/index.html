Projeto Controle de Funcionários

API de Serviços baseados em REST (WebApi)
  • POST - /api/funcionario/cadastro
  • Descrição: Cadastrar um novo funcionário
  • Tipo de dados: JSON
  • Dados de entrada:
    • Nome - Texto
    • Cargo - Texto
    • Email - Texto
  • Saída HTTP:
    • 200 - OK (Funcionário cadastrado com sucesso)
    • 403 - Erros de validação dos dados
    • 400 - Erro do servidor
  • PUT - /api/funcionario/editar
  • Descrição: Editar dados de um funcionário
  • Tipo de dados: JSON
  • Dados de entrada:
    • FuncionarioId - Inteiro
    • Nome - Texto
    • Cargo - Texto
    • Email - Texto
  • Saída HTTP:
    • 200 - OK (Funcionário atualizado com sucesso)
    • 403 - Erros de validação dos dados / funcionário não encontrado
    • 400 - Erro do servidor
  • DELETE - /api/funcionario/excluir
  • Descrição: Excluir um funcionario
  • Tipo de dados: JSON
  • Dados de entrada:
    • FuncionarioId - Inteiro
  • Saída HTTP:
    • 200 - OK (Funcionário excluido com sucesso)
    • 403 - Funcionário não encontrado
    • 400 - Erro do servidor
  • GET - /api/funcionario/obter
  • Descrição: Obter dados de 1 Funcionário
  • Tipo de dados: JSON
  • Dados de entrada:
    • FuncionarioId - Inteiro
  • Saída HTTP:
    • 200 - OK (Objeto contendo os dados do Funcionário)
    • 403 - Funcionário não encontrado
    • 400 - Erro do servidor
  • GET - /api/funcionario/listar
  • Descrição: Listar todos os Funcionários
  • Tipo de dados: JSON
  • Saída HTTP:
    • 200 - OK (Lista de objetos contendo os dados dos Funcionários)
    • 403 - Nenhum Funcionário encontrado
    • 400 - Erro do servidor
